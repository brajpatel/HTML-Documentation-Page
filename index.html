Introduction

HTML stands for Hyper Text Markup Language and is used to describes how a web page should be structured. HTML elements are the building blocks of HTML documents which web browsers receive from a web server or local storage.

We use a start tag, some content and an end tag (< >"content"</>) to define each element.

Here are some commonly used HTML tags:

  • <HTML>...</HTML> - the entire HTML document
  • <head>...</head> - the head of the document
  • <title>...</title> - specifies a title for the HTML page shown in the browsers tab
  • <body>...</body> - where the web pages content goes
  • <h1-h6>...</h1-/h6> - levels of heading
  • <p>...</p> - paragraph, create a new tag for each paragraph
  • <br> - line break

Whether you want to learn to code to learn something new, or for career purposes, HTML is a great place to get yourself started.

To create your very own web page, all you need is a web browser (E.g. Chrome) and a simple text editor like Notepad (PC) or TextEdit (Mac).

HTML Documents

All documents must be have a <!DOCTYPE> declaration (only appears once at top of page). This represents the document type and helps the browsers to display the web page correctly. The <html> element is the root element of the HTML page.

Here's what the declaration for HTML5 looks like:

<!DOCTYPE html>
<html>

</html>

To save your web page, it must have the file extension .htm or .html. There is no difference between them, both work.

HTML Headings

Headings are useful for showing the starting point of a section or topic and are defined by the <h1>> to <h6> tags.

<h1> is the heading with the most importance and should only be used once in a web page. Subsequent headings having less importance all the way down to <h6>.

<h1>This is the most important heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is least important heading 6</h6>
HTML Paragraphs

The <p> tag is used to define HTML paragraphs. Anything between the opening <p> tag and ending </p> tag is part of the paragraph.

A paragraph always starts on a new line with browsers automatically adding a small margin before and after it.

<p>This is a paragraph.</p>
<p>This is also a paragraph.</p>
Hello World

We've already covered enough for you to be able to write your very first lines of HTML code for your web page. Here's what a simple HTML document looks like:

<!DOCTYPE html>
<html>

<head>
<title>Hello World</title>
</head>

<body>

<h1>My first heading</h1>
<p>My first paragraph</p>

<p>My second paragraph</p>

</body>
</html>

After saving your work using the .htm or .html file extensions, open it up in your browser to see what youve created.

HTML Links

Links are added to web pages using the <a> tag. Just like this one. We can specify the destination of the link by using the href attribute or make the link dead by adding a '#'.
Attributes are used to provide additional information about a HTML element.

We can open and close tags within other tags. Here's what that might look like:

<p>This paragraph has a <a href="#">link</a> in it.</p>

This paragraph has a link in it.
HTML Images

Images in HTML are defined using the <img> tag. This tag is self-closing, meaning a separate opening and closing tag is not necessary.

The src attribute lets us specify the source file. And the alt attribute (alternative text) lets us choose the text viewers see should the image fail to load properly. We can also change the height and width of the image we want to use so it fits better in our web page.

Here's what this might look like:

<img src="happy-puppy.jpg" alt="A happy puppy" width="150px" height="188px">



View HTML Source

Have you ever seen a web page and wondered, "How exactly have they done that?".

View HTML source code:

Right click on a web page and select "view page source" (in chrome) or similar in other browsers. This will open a window that contains the HTML source code of that page.

Inspect a HTML element:

Right click on an element or blank area and select "Inspect" or "Inspect element" (or similar). You'll be able to see what elements are made of and also how they have been styled using CSS (Cascading Style Sheets). You can also edit parts of the HTML and CSS on-the-fly in the panel that opens.
